MoveSphere

Use:
Open the scene MoveSphere, play it, click in the game window.
It is a simple scene with a sphere moving (right/left and up/down)
at the "wasd" keys or arrow keys.

Important features used (see the ExMoveSphere graph):
1. Get Axis: get the Horizontal and the Vertical input axis.
2. Manipulate Vector3.
3. Set Velocity on Rigidbody.
4. Make a public variable.
