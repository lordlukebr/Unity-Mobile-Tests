AtStateStart
Use:
Open the scene ExAtStateStart, play it, click in the game view, 
press the 'C' key to switch to the counter state and to count to 32 on the screen.
Press the 'R' key to return to the Idle state and to return to zero on the screen.

Important features used (see the ExAtStateStart graph):
1. At State Start, offering a user defined number of true pulses 
	when a state is begening.
